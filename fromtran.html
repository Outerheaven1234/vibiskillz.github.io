Section 1: Intro: Cheating our way past the transistor

A basic understanding of transistors:
Transistors use current(BJT) or voltage(MOSFET) to regulate the current influx through a circuit. This is especially convenient for basic operations such as logic gates.

Basic logic gates using transistors:
Using resistors, npn transistors, and input sources, I can replicate the properties of various logic gates. Using LTSpice I simulate a nand gate using 2 npn transistors and output the values of the output voltage based on the 2 input voltages. As you can see the output voltage is low whenever input A and B are high. Using this theory for basic logic gates i can create even more complex circuits to represent more high functioning logic tasks.

What is a mux?
A mux, or multiplexer uses logic to pass input values to the output depending on selector values. Using LTSpice i simulated a circuit to represent a 2 x 1 mux using and gates, or gates, and inverters. Muxes simplify the process of passing output values using selectors and is a core design concept in engineering.

What is a latch?
A latch uses logic to store data in the form of a 0 or 1. Using a set and reset switch, the values being stored can be manipulated. Latches come in 2 distinct forms: active-high and active-low. Active-high SR latches switch states when a 1 or high value is set while active-low switches states when a 0 or low value is set. Logically, an active-high SR latch uses NOR gates and an active-low SR latch uses NAND gates.

What are flip flops?
Flip flops are essentially latches(circuits/logic gates) that store states. The potential values for these latches can be found through truth/lookup tables. There are 2 primary types of flip flops, RS flip-flops and D flip-flops. RS flip-flops are unclocked and are esenetially the same as latches(as mentioned above). Whereas D flip=flops are clocked latches that use a consistant square wave and a second set of gates nand or nor gates(whether its acitve high or low). The purpose of a D flip-flop is to ensure that R and S are never equal at any point in time using a clock pulse.
